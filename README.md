# Helmet Detection and License Plate Recognition

## Overview
This project detects whether bike riders are wearing helmets and recognizes vehicle license plates using computer vision and deep learning techniques.

## Features
- Helmet detection using image processing
- License plate recognition
- Real-time monitoring support
- Traffic rule violation detection

## Technologies Used
- Python
- OpenCV
- YOLO
- NumPy

## Project Workflow
1. Input image/video capture
2. Helmet detection
3. License plate extraction
4. Character recognition
5. Output generation

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
