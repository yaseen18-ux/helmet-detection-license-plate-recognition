# Helmet Detection and License Plate Recognition

This project is a computer vision–based traffic monitoring system that detects helmet violations and identifies vehicle license plates from video input. The system processes video frames in real time, detects riders, analyzes helmet usage, and identifies number plates for monitoring and safety enforcement purposes.

## Project Objective

The objective of this project is to develop an automated traffic monitoring system capable of:

- Detecting motorcycle riders
- Identifying helmet violations
- Detecting vehicle license plates
- Processing video frames in real time
- Supporting road safety monitoring and enforcement

## Tools & Technologies Used

- Python
- OpenCV
- NumPy
- Imutils
- Haar Cascade Classifier
- HOG Descriptor
- Computer Vision Techniques

## Project Features

The system includes the following functionalities:

- Helmet detection
- Person detection
- Face detection
- Number plate detection
- Real-time video processing
- Violation identification
- Detection summary display
- Output video generation

## System Workflow

The project follows the below workflow:

1. Capture video input
2. Process video frames
3. Detect riders and persons
4. Analyze helmet usage
5. Detect number plate regions
6. Display detection results
7. Save processed output video
